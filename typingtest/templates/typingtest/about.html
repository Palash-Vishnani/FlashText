About Us

FlashText.com

FlashText is a website where a user can practice and test his/her typing skills. Also, they can create account to store their data and statistics, so that they can get a clear picture about their progress. This website will also give you tips on how to type faster, also we have a Contact us page which makes the website more interactive, in case our users have any query or feedback for us. We consider this website as one of our major project which has basic functions with both Front End and Back End functioning properly. We've uploaded this website as a project on Github, so that anyone can change or add features and functionalities to make this website more user friendly.
